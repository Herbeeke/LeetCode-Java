给定两个大小分别为 `m` 和 `n` 的正序（从小到大）数组 `nums1` 和 `nums2`。请你找出并返回这两个正序数组的 **中位数** 。

算法的时间复杂度应该为 `O(log (m+n))` 。

**示例 1：**

```
输入：nums1 = [1,3], nums2 = [2]
输出：2.00000
解释：合并数组 = [1,2,3] ，中位数 2

```

**示例 2：**

```
输入：nums1 = [1,2], nums2 = [3,4]
输出：2.50000
解释：合并数组 = [1,2,3,4] ，中位数 (2 + 3) / 2 = 2.5

```

**提示：**

- `nums1.length == m`
- `nums2.length == n`
- `0 <= m <= 1000`
- `0 <= n <= 1000`
- `1 <= m + n <= 2000`
- `10^6 <= nums1[i], nums2[i] <= 10^6`

**优秀题解：**

```java
class Solution {
    // 思路或算法：使用了二分查找的方法，时间复杂度为 O(log(min(m, n)))，其中 m 和 n 分别是两个输入数组的长度
    public double findMedianSortedArrays(int[] nums1, int[] nums2) {
        // 将nums1设为较短的数组，以减少二分查找的时间复杂度
        if (nums1.length > nums2.length) {
            int[] temp = nums1;
            nums1 = nums2;
            nums2 = temp;
        }

        int m = nums1.length;
        int n = nums2.length;

        // 设置二分查找的左边界和右边界
        int left = 0;
        int right = m;

        // 计算两个数组的总长度，用于判断中位数的位置
        int totalLength = m + n;

        // 使用二分查找来寻找中位数
        while (left <= right) {
            int partition1 = (left + right) / 2; // 在nums1数组中进行二分查找的分割点
            int partition2 = (totalLength + 1) / 2 - partition1; // 在nums2数组中进行二分查找的分割点

            // 计算分割点左右两侧的值
            int maxLeft1 = (partition1 == 0) ? Integer.MIN_VALUE : nums1[partition1 - 1];
            int minRight1 = (partition1 == m) ? Integer.MAX_VALUE : nums1[partition1];
            int maxLeft2 = (partition2 == 0) ? Integer.MIN_VALUE : nums2[partition2 - 1];
            int minRight2 = (partition2 == n) ? Integer.MAX_VALUE : nums2[partition2];

            // 判断是否找到中位数
            if (maxLeft1 <= minRight2 && maxLeft2 <= minRight1) {
                // 如果总长度为偶数，则中位数为两个分割点左侧的最大值和右侧的最小值的平均值
                if (totalLength % 2 == 0) {
                    return (Math.max(maxLeft1, maxLeft2) + Math.min(minRight1, minRight2)) / 2.0;
                }
                // 如果总长度为奇数，则中位数为两个分割点中的较大值
                else {
                    return Math.max(maxLeft1, maxLeft2);
                }
            } else if (maxLeft1 > minRight2) {
                // 如果nums1的分割点在左侧偏大，则将分割点左移
                right = partition1 - 1;
            } else {
                // 如果nums1的分割点在左侧偏小，则将分割点右移
                left = partition1 + 1;
            }
        }

        return 0.0; // 如果数组不是正序的，则返回0.0
    }
}
```
